Secure Links

When you create a link for your HTML site you typically set the "target" attribute to "_blank", however without additional attributes you will be subject to "tabnabbing!" This is when bad actors gain access to window.object API and redirect to a "fake page." This typically happens with login pages. Since you are opening in a new tab most users would not know this happened. To prevent this add "rel = "noopener noreferr" within your link attributes. Example: < a href = "https://id.heroku.com/" target = "_blank" rel = "noopener noreferr">Heroku Login Page< /a> Instead of being redirected to a fake Heroku Login Page you will go to the actual login page.

Capital "T" Like Spongebob

If you have a large block of text and want the first letter to be larger than the rest use "DROPCAP". In CSS type .dropcap and make one of the attributes 'font-size = 400%'. This will make that letter bigger than the rest. You would wrap your letter in a < span class = "dropcap"> T < /span>

Shortcut Console.Log

You can use VAR or CONST for "console.log" so that you have a "shorcut" each time you want to console.log something. Example: 'var c = console.log.bind(document)' if your code includes c(VeggieDelight) // output = VeggieDelight.
